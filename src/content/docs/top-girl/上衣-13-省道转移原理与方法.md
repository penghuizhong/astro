## 一、课程概述

本节课系统讲解**省道转移**的原理与分类，是上衣制版从原型走向款式变化的核心基础课。

省道的处理方式从大类上分为两种：

| 大类 | 说明 |
|------|------|
| **省道转移** | 省道的位置与形态发生改变，省量等量转移至其他位置，省量总值不变 |
| **省道消除** | 省量通过归拔、吃缝、装饰褶等工艺手段分散隐藏于结构中，不以省道形式显现，但省量依然存在 |

> **本节重点讲解省道转移。**

---

## 二、省道转移的三种类型

按转移的数量关系，省道转移分为三种：

| 类型 | 说明 | 典型应用 |
|------|------|---------|
| **一转一** | 一条省道等量转移至另一位置，数量不变 | 胸省转腰省、胸省转肩省 |
| **一转多** | 一条省道分散转移为多条省道或多个褶 | 碎褶、放射褶、分割缝 |
| **多转一** | 多条省道集中合并为一条省道 | 六省转四省、腰省合并 |

---

## 三、省道转化后的四种形态

省道转移后，在成衣上可以表现为以下四种形态：

### 3.1 省道形态（Province）

转移后仍以**省道**形式存在，位置与原省不同。

### 3.2 缝道形态（Seam）

将原本完整的一片裁片在省道位置**破开**，分成两片，再拼合缝制。两片缝合后，缝道承担了原省道的造型功能。

> **理解要点：** 一条缝 = 一条省（功能等价），区别在于省是在一片上折叠，缝是将一片破开后再拼合。

### 3.3 褶裥形态（Pleat / Gather）

省量以**褶裥**形式处理，包括：

| 褶裥类型 | 特征 |
|---------|------|
| 单向褶 | 省量倒向一侧 |
| 对褶（工字褶） | 省量向两侧对称展开 |
| 碎褶 / 抽褶 | 省量均匀分散为细密褶皱 |

### 3.4 工艺量形态（Ease / Fullness）

省量通过**归拔、吃缝**等工艺手段处理，隐藏于成品中，肉眼不易察觉。这是最隐蔽的处理形式。

> **省量不会消失，只会换一种形式存在。** 无论以哪种形态出现，总省量守恒。

---

## 四、省道转移的方法——指向剪接法

本节采用**指向剪接法**（又称旋转剪接法）进行省道转移，操作步骤如下：

1. 在原型上确定**省道转移的目标位置**（新省所在位置）
2. 从 BP 点出发，向目标位置做辅助切割线
3. 合并原省道（将原省道缝合量折叠），原省随之打开
4. 以 BP 点为旋转中心，旋转裁片，使原省合拢、目标位置打开
5. 打开量即为转移后的新省量，描净轮廓线完成转移

> **本质：** 指向剪接法就是一种旋转拼接操作，与之前四省转化中将省道打入袖窿的操作原理完全相同，只是目标位置不同。

## 五、胸省的360°转移位置与命名

胸省（以 BP 点为中心）可以向前片任意方向转移，不同位置对应不同名称：

| 转移目标位置 | 省道名称 |
|------------|---------|
| 袖窿处 | **袖窿省** |
| 肩部 | **肩胸省** |
| 领口处 | **领胸省** |
| 门襟处（前中线） | **撇胸**（门襟撇胸 / 门金省） |
| 腰部 | **腰胸省**（与腰省合并后成为复合省量） |
| 腋下 | **腋下省** |
| 平行于胸围线 | **横胸省（旁胸省）** |

> **撇胸的特殊说明：** 撇胸是将胸省转移至前中线（门襟）方向，使前止口线产生倾斜，是外套、大衣常见的处理方式，与门金省（门襟省）属同一原理。

## 六、一转一操作示例

### 6.1 胸省转袖窿省

这是最基础的一转一操作，也是四省原型制作中已练习过的方法：

1. 取出合身圆形（原型）
2. 确认原胸省位置（袖窿斜线方向）
3. 将原胸省合并（折叠），目标位置（袖窿）相应打开
4. 量取打开量：约 **0.0 几 cm**（极小），几乎不影响版型，可忽略不计

> 转移后，以软件"折量工具"处理褶倒向：鼠标方向即为折量倒向方向，向上倒则底部增量，向下倒则顶部增量。

---

## 七、一转多操作示例

### 7.1 胸省转多条碎褶 / 放射褶

将一条胸省分散转移为多条省道（或细褶），步骤如下：

1. 从 BP 点向目标位置（如肩线）做多条辅助切割线，将前片分成多个扇形区域
2. 依次合并原胸省，每个扇形区域对应打开等比例的量
3. 所有打开量之和 = 原省总量

> **规律：** 所有辅助切割线必须交汇于 **BP 点**（或距 BP 点固定距离的一个公共点），才能形成放射状均匀分布的褶裥。

### 7.2 平行褶处理

若需做**两条平行褶**（不汇聚于 BP 点）：

1. 找到两条平行褶线共同的**等量交汇点**（两线延长后的交点，作为辅助旋转中心）
2. 依次从该点出发做切割，分别打开等量
3. 两条平行褶的打开量必须相等，否则缝合后边长不一，无法拼合

> **注意：** 平行褶不能各自完全指向 BP 点，但必须通过"共同交汇点"来保证两条褶边长一致。

## 八、多转一操作示例

### 8.1 多条省合并为一条腰省

以前片多条省（胸省 + 腰省）合并为一条腰省为例：

1. 从 BP 点向腰线做辅助切割线，确定新省中心位置
2. 将所有参与合并的省道依次旋转至腰线方向
3. 合并后的腰省量 = 胸省量 + 原腰省量（复合省量）

> **省量叠加公式：**
> $$\text{合并后腰省量} = \text{胸省量} + \text{腰省量}$$
> 合并后形成一条较大的腰省，同时兼具胸部与腰部的造型功能。

### 8.2 省尖位置处理

- 省尖不宜打到距离 BP 点太近的位置，否则省道收尖处过于突出，产生"锥形"视觉效果
- 通常省尖距 BP 点保留 **1.5～2.5 cm** 的距离（原型中后腰省也不打到臀围线，同理）
- 若需要省道完全交于 BP 点（如做某些特殊设计），可先将省尖打到 BP 点用于旋转操作，操作完成后再将省尖退回至合适距离

## 九、省道转为缝道操作示例

### 9.1 省转缝的条件

省道转为缝道（分割缝），**必须满足以下条件**：

$$\text{分割缝线通过 BP 点（或经过 BP 点附近 ≤ 1.5 cm）}$$

若分割缝不经过 BP 点，则缝道无法完整承担省量，成品会在缝道周围出现**容位（浮量）**，即布料松散不服帖的区域。

### 9.2 操作步骤

1. 确定分割缝位置，使其通过（或经过）BP 点
2. 在 BP 点处将前片**沿分割缝线破开**，分为两片
3. 合并原胸省，使分割缝两侧自然展开，形成缝道造型量
4. 分别修顺两片的缝道边线，确保两侧边长一致（缝合后等长）

### 9.3 有容位的缝道处理

若分割缝**不经过 BP 点**（距 BP 点有一定距离），缝道将带来容位：

- 容位量约为 **0.5 cm**（视距离远近而定）
- 在样板上**标注归缩符号**（波浪线），注明归缩量：约 **0.5 cm**
- 在对应缝合位置**打刀口**，告知车缝工人此处需吃缝归缩

> **刀口位置：** 在容位起止处各打一个刀口，车缝时在两刀口之间的范围内均匀归缩 0.5 cm，使缝合后两边等长。

## 十、省道转移的注意事项

### 10.1 省量守恒原则

$$\text{转移前总省量} = \text{转移后总省量}$$

无论转移到哪个位置、以何种形态出现，总量不变。这是省道转移的核心原则。

### 10.2 纱向原则

省道转移完成后，各裁片的纱向线（经纱方向）必须垂直于地面。若转移后纱向偏斜，需通过整体旋转修正，或重新确认转移路径。

### 10.3 省量大小与形态的关系

- 省量较小（≤ 1 cm）：适合转为工艺量（归拔处理），肉眼难以察觉
- 省量中等（1～3 cm）：适合转为缝道或单向褶
- 省量较大（≥ 3 cm）：适合转为对褶、碎褶，或通过分割缝处理

### 10.4 省存量与放松量的关系

腰节差加放 0.5 cm 省存量的意义：

$$\text{省存量（0.5 cm）} = \text{人体活动所需的最小松量}$$

大廓形款式（如大衣）胸围放松量可达 **20～30 cm** 甚至更多，此时各部位所需的转折量（即结构用量）显著增大，若省量或转折量不足，结构线会出现"走丝"或不顺现象，这部分内容将在后续转折量课程中详细讲解。
