# 裙子基础版型

## 结构组成

裙子版型由以下主要裁片组成：
- 前裙片
- 后裙片
- 腰头
- 开衩/拉链位

## 关键尺寸

| 部位 | 测量方法 | 备注 |
|------|----------|------|
| 腰围 | 腰部最细处水平围量 | 加放 2cm |
| 臀围 | 臀部最丰满处水平围量 | 加放 4cm |
| 裙长 | 腰至下摆 | 根据款式调整 |
| 臀高 | 腰至臀围线 | 约 18-20cm |

## 绘制要点

1. 确定基础框架（臀围线、裙长线）
2. 绘制前后裙片轮廓
3. 处理省道（前后各 2 个）
4. 绘制下摆线
5. 添加缝份与开衩位
