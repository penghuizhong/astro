# 裤子基础版型

## 结构组成

裤子版型由以下主要裁片组成：
- 前裤片
- 后裤片
- 腰头
- 口袋（可选）

## 关键尺寸

| 部位 | 测量方法 | 备注 |
|------|----------|------|
| 腰围 | 腰部最细处水平围量 | 加放 2-3cm |
| 臀围 | 臀部最丰满处水平围量 | 加放 4-6cm |
| 裤长 | 腰侧点至脚踝 | 根据款式调整 |
| 立裆深 | 腰至裆底 | 坐姿测量更准 |

## 绘制要点

1. 确定基础框架（臀围线、横裆线）
2. 绘制前裤片（小裆弯）
3. 绘制后裤片（大裆弯）
4. 处理省道与褶裥
5. 添加缝份与腰头
