# 上衣基础版型

## 结构组成

上衣版型由以下主要裁片组成：
- 前片
- 后片
- 袖子
- 领子

## 关键尺寸

| 部位 | 测量方法 | 备注 |
|------|----------|------|
| 胸围 | 胸部最丰满处水平围量 | 加放 4-6cm |
| 肩宽 | 左肩点至右肩点 | 自然状态测量 |
| 衣长 | 颈侧点至下摆 | 根据款式调整 |
| 袖长 | 肩点至腕点 | 手臂微屈状态 |

## 绘制要点

1. 先确定基础框架（胸围线、腰节线）
2. 绘制前后片轮廓
3. 处理省道（胸省、腰省）
4. 绘制袖窿弧线
5. 添加缝份
