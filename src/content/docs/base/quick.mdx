## 前置要求

- 已注册的方圆智版 AI 账号
- 现代浏览器（推荐 Chrome / Edge / Safari）

## 第一步：进入工作台

点击首页的「进入工作台」按钮，或直接访问 `/workbench`。

## 第二步：创建新项目

1. 点击「新建项目」
2. 选择版型模板（上衣 / 裤子 / 裙子）
3. 输入基础尺寸数据

## 第三步：AI 辅助生成

上传设计稿或输入参数，AI 将自动生成基础版型。
