# AI 辅助制版流程

## 流程概览

```
设计输入 → AI 分析 → 版型生成 → 人工调整 → 放码输出
```

## 详细步骤

### 1. 设计输入

支持以下输入方式：
- 手绘草图上传
- 技术参数输入（胸围、腰围、衣长等）
- 参考图片分析

### 2. AI 分析

AI 引擎会：
- 识别款式特征（领型、袖型、省道位置）
- 推算尺寸比例关系
- 匹配最接近的基础版型

### 3. 版型生成

自动生成包含以下要素的版型：
- 裁片轮廓
- 省道位置与大小
- 缝份预留
- 对位标记

### 4. 人工调整

在交互式编辑器中：
- 拖拽调整关键点
- 实时查看尺寸变化
- 预览 3D 效果

### 5. 放码输出

一键生成全码版型，支持：
- 标准码（S/M/L/XL）
- 自定义尺码表
- DXF/AAMA 格式导出
